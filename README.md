
# Amazon Product Scraper

A Python-based web scraping tool that collects product information from Amazon search pages.  
This project is **currently in progress** and is for **educational purposes only**.

## Features (Work in Progress)
- Extract product names ✅
- Extract prices ✅
- Extract ratings ✅
- Save collected data to a CSV file (coming soon)

## Technologies Used
- Python
- BeautifulSoup
- Requests
- Pandas

## Purpose
This project is being developed as a learning exercise to understand:
- Web scraping and data extraction
- Automation with Python
- Handling and saving structured data

## Usage
1. Make sure you have Python installed (3.10+ recommended)
2. Install required libraries:

```bash
pip install requests beautifulsoup4 pandas
